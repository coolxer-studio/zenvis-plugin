# 用户事件菜单配置说明文档

## 菜单概述

该配置文件定义了用户事件相关的两个功能菜单项，分别指向不同的功能模块和页面，为用户提供数据可视化和分析能力。

## 菜单项详情

### 1. 用户事件可视化

| 属性 | 值 | 说明 |
|------|----|------|
| 菜单名称 | 用户事件可视化 | 用于展示用户事件数据的可视化界面 |
| 路由地址 | `policy-config` | 系统内的路由标识 |
| 参数 | `com.coolxer.plugin.user.event` | 传递给页面的参数，标识插件命名空间 |
| 类型 | `4` | 菜单类型标识，对应可视化配置页面 |

### 2. 用户事件分析

| 属性 | 值 | 说明 |
|------|----|------|
| 菜单名称 | 用户事件分析 | 用于分析用户事件数据的低代码页面 |
| 路由地址 | `low-code-page` | 系统内的路由标识，指向低代码平台页面 |
| 参数 | `com.coolxer.plugin.user.event` | 传递给页面的参数，标识插件命名空间 |
| 类型 | `3` | 菜单类型标识，对应低代码配置页面 |

## 功能说明

### 用户事件可视化
该菜单项主要提供用户事件数据的图形化展示功能，可能包括：
- 数据统计图表
- 趋势分析视图
- 实时监控面板
- 数据分布可视化

### 用户事件分析
该菜单项提供基于低代码平台的用户事件数据分析功能，可能包括：
- 自定义数据查询
- 多维度数据分析
- 报表生成与导出
- 数据筛选与聚合分析

## 技术说明

两个菜单项都使用相同的参数值 `com.coolxer.plugin.user.event`，表明它们属于同一个插件模块的不同功能页面。类型值的不同（`3`和`4`）表示它们指向系统中不同类型的页面模板。

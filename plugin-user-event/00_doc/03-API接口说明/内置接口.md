# user-event 调试信息实体接口文档

## 概述

本文档详细描述了 `user-event` 实体的 RESTful API 接口。该实体用于记录和管理系统中的用户行为事件信息，主要用于测试验证场景。

### 基本信息

- **实体名称**: user-event（调试信息）
- **数据表名**: msg_user_event
- **数据源**: ClickHouse
- **存储引擎**: MergeTree()
- **排序字段**: id, server_time
- **分区策略**: 按月份分区 (toYYYYMM(server_time))

## 统一响应格式

所有接口返回统一的 JSON 格式：

```json
{
  "status": 0,        // 状态码，0表示成功
  "msg": "消息描述",   // 响应消息
  "data": {}          // 响应数据
}
```


## 接口列表

### 1. 添加用户事件

**接口地址**: `POST /api/v1/entity/user-event/add`

**功能说明**: 创建一个新的用户事件记录

**请求参数**:
- 请求体: 用户事件数据 JSON 对象，包含以下字段：
    - `id` (String): 事件ID
    - `procid` (Integer): 进程ID
    - `user` (String): 用户名
    - `event_type` (String): 事件类型
    - `reliability` (Float): 可信度
    - `detail` (JSON): 数据详情
    - `tags` (Array[String]): 标记
    - `server_time` (DateTime): 入库时间

**响应示例**:
```json
{
  "status": 0,
  "msg": "添加成功",
  "data": null
}
```

### 6. 查询用户事件列表

**接口地址**: `GET /api/v1/entity/user-event/list`

**功能说明**: 分页查询用户事件列表，支持多种筛选条件

**请求参数**:
- 查询参数: 过滤条件键值对，支持以下字段：
    - [id](file://cn/hutool/core/lang/tree/TreeNode.java#L6-L6): 事件ID (equal, notequal, in)
    - `procid`: 进程ID (greatthan, lessthan, greatequalthan, lessequalthan, between)
    - [user](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/web-service/src/main/java/com/coolxer/model/system/vo/LoginVo.java#L23-L23): 用户名 (equal, notequal, in)
    - `event_type`: 事件类型 (equal, notequal, in)
    - `reliability`: 可信度 (equal, notequal, greatthan, lessthan, greatequalthan, lessequalthan, between)
    - [tags](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/web-service/src/main/java/com/coolxer/model/retrieval/vo/DeviceInfoVo.java#L24-L24): 标记 (in)
    - `server_time`: 入库时间 (greatthan, lessthan, greatequalthan, lessequalthan, between)

**响应示例**:
```json
{
  "status": 0,
  "msg": "请求成功",
  "data": {
    "rows": [
      {
        "id": "event_001",
        "procid": 12345,
        "user": "testuser",
        "event_type": "login",
        "reliability": 0.95,
        "detail": {"action": "login_success"},
        "tags": ["auth", "login"],
        "server_time": "2023-01-01 12:00:00.000"
      }
    ],
    "total": 100
  }
}
```

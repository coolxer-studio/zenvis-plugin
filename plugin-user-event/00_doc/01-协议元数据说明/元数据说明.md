## 操作符定义

| ID | 名称 | 标签 |
|----|------|------|
| 1 | `equal` | 等于 |
| 2 | `notequal` | 不等于 |
| 3 | `greatthan` | 大于 |
| 4 | `lessthan` | 小于 |
| 5 | `greatequalthan` | 大于等于 |
| 6 | `lessequalthan` | 小于等于 |
| 7 | `between` | 之间 |
| 8 | `in` | 包括 |

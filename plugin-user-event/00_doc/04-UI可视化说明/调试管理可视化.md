# 调试管理UI界面说明文档

## 界面概述

这是一个基于低代码平台构建的调试信息管理界面，用于展示和管理用户事件数据。界面提供数据查看、搜索、创建、编辑和删除等完整功能。

## 页面基本信息

- **页面类型**: [page](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/plugin/extend-api/src/main/java/com/coolxer/plugin/controller/UserSearchDto.java#L45-L45)
- **标题**: 调试管理
- **主要功能**: 用户事件数据的增删改查操作

## 工具栏功能

### 创建记录按钮

在页面顶部工具栏提供"创建记录"按钮，点击后弹出创建对话框：

- **按钮类型**: `button`
- **触发动作**: 弹出对话框
- **标签**: 创建记录
- **样式**: 主要按钮样式

#### 创建对话框配置

- **标题**: 创建记录
- **布局模式**: 水平布局
- **提交接口**: `/zenvis/api/v1/entity/user-event/add`
- **校验规则**: 可信度值不能大于10.00

##### 表单字段

1. **ID**
    - 类型: UUID自动生成
    - 显示: 静态文本显示生成的ID

2. **进程ID** (`procid`)
    - 类型: 文本输入框
    - 必填项: 是

3. **用户名** ([user](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/plugin/extend-api/src/main/java/com/coolxer/plugin/controller/UserVo.java#L16-L16))
    - 类型: 文本输入框
    - 必填项: 是

4. **事件类型** (`event_type`)
    - 类型: 下拉选择框
    - 数据源: `/zenvis/api/v1/entity/user-event/event_type/mapping`

5. **可信度** (`reliability`)
    - 类型: 文本输入框
    - 必填项: 是
    - 校验: 值不能大于10.00

6. **数据详情** (`detail`)
    - 类型: 多行文本框
    - 必填项: 是

7. **标记** (`tags`)
    - 类型: 标签输入框
    - 数据源: `/zenvis/api/v1/entity/user-event/tags/list`
    - 支持清除: 是
    - 必填项: 是

8. **入库时间** (`server_time`)
    - 类型: 日期时间选择器
    - 格式: YYYY-MM-DD HH:mm:ss
    - 默认值: 当前时间
    - 必填项: 是

## 数据展示区域

使用CRUD组件展示用户事件数据列表：

- **数据接口**: `/zenvis/api/v1/entity/user-event/list`
- **快速保存接口**: `/zenvis/api/v1/entity/user-event/$id/update`
- **自动过滤**: 启用自动生成功能

### 数据列说明

1. **事件ID** ([id](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/plugin/extend-api/src/main/java/com/coolxer/plugin/controller/UserVo.java#L15-L15))
    - 显示格式: 截取前14个字符
    - 功能: 支持复制
    - 可搜索: 是

2. **进程ID** (`procid`)
    - 可搜索: 是

3. **用户** ([user](file:///Users/yaoqi.li/Documents/cabin/coolxer/zenvis/plugin/extend-api/src/main/java/com/coolxer/plugin/controller/UserVo.java#L16-L16))
    - 搜索功能: 提供自动补全
    - 自动补全接口: `/zenvis/api/v1/entity/user-event/user/auto-complete?term=$term`

4. **标签** (`tags`)
    - 显示格式: 以标签形式展示
    - 快速编辑: 支持直接编辑
    - 搜索功能: 下拉选择搜索
    - 数据源: `/zenvis/api/v1/entity/user-event/tags/list`

5. **事件类型** (`event_type`)
    - 显示格式: 根据类型显示不同颜色标签
        - 登录: 蓝色标签
        - 点击: 蓝色标签
        - 浏览: 蓝色标签
        - 删除: 橙色标签
        - 修改: 橙色标签
        - 其他: 显示原始值
    - 搜索功能: 下拉选择搜索
    - 数据源: `/zenvis/api/v1/entity/user-event/event_type/mapping`

6. **可信度** (`reliability`)
    - 可搜索: 是

7. **入库时间** (`server_time`)
    - 搜索功能: 时间范围选择器
    - 默认显示: 是

8. **详情** (`detail`)
    - 显示格式: 截取前20个字符
    - 悬停查看: 鼠标悬停可查看完整JSON详情
    - 默认显示: 否

### 操作列

提供编辑和删除操作按钮：

1. **编辑按钮**
    - 图标: 铅笔图标
    - 动作类型: 弹出对话框
    - 对话框大小: 大尺寸
    - 编辑表单字段与创建表单基本一致，但不包含数据详情和入库时间字段

2. **删除按钮**
    - 图标: 红色叉号图标
    - 动作类型: AJAX请求
    - 确认提示: 删除前需要用户确认
    - 删除接口: `delete:/zenvis/api/v1/entity/user-event/$id`

## 搜索功能

界面支持多种搜索方式：
- 文本字段支持直接输入搜索
- 下拉字段支持选择搜索
- 时间字段支持范围选择搜索
- 标签字段支持多选搜索
- 用户字段支持自动补全搜索

## 特色功能

1. **快速编辑**: 部分字段(如标签)支持在列表中直接编辑并立即保存
2. **数据校验**: 创建和编辑时对可信度字段进行校验
3. **可视化展示**: 不同事件类型使用不同颜色标签展示
4. **详情查看**: 数据详情字段支持悬停查看完整内容
5. **自动补全**: 用户名字段支持搜索自动补全功能
